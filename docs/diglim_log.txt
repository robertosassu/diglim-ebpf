NAME
diglim_log - DIGLIM eBPF logging service


SYNOPSIS
diglim_log


DESCRIPTION
diglim_log takes the logs produced by the eBPF program and forwards them to
syslog (e.g. systemd journal).


EXAMPLES
Start the DIGLIM eBPF logging service:

# diglim_log
